## Field descriptions

- **Basic**: basic FEL communication (version, memory read/write, execute);
- **Reset**: watchdog reset support;
- **Sid**: SID reading support;
- **Jtag**: JTAG debug enable support;
- **DDR**: DDR controller initialization support;
- **SPI Nor Flash**: SPI NOR Flash operation support;
- **SPI Nand Flash**: SPI NAND Flash operation support.
