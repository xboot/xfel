# DDR 类型参考

`xfel ddr [type]` 命令支持的可选类型因芯片而异。下表列出各芯片支持的 DDR 类型名称，未列出的芯片使用默认配置（无需指定 `type`）。

| 芯片 | 支持的 DDR 类型 |
|------|----------------|
| D1 / F133 | `d1`、`f133` |
| H2 / H3 | `h2`、`h3` |
| H616 / H313 / H618 | `lpddr4` |
| V3s / S3 | `v3s`、`s3` |
| V851 / V853 | `v851`、`v853` |
| R528 / T113 | `r528-s3`、`t113-s3`、`t113-s4` |
| R128 | `r128-s2`、`r128-s3`、`xr875` |
| T153 | `t153-m3` |
| A40i / R40 | `ddr3`、`lpddr3` |
| A523 / A527 / T527 | `lpddr4` |

## 用法示例

```shell
# D1 / F133
xfel ddr f133

# R528-S3
xfel ddr r528-s3

# A523 系列，使用 LPDDR4
xfel ddr lpddr4
```

:::note
若不确定具体类型，可直接执行 `xfel ddr` 使用芯片默认配置。当板载内存与默认配置不符时，才需显式指定类型。
:::
