## еӯ—ж®өиҜҙжҳҺ

- **Basic**пјҡеҹәзЎҖ FEL йҖҡдҝЎиғҪеҠӣпјҲзүҲжң¬иҜҶеҲ«гҖҒеҶ…еӯҳиҜ»еҶҷгҖҒжү§иЎҢпјүпјӣ
- **Reset**пјҡзңӢй—ЁзӢ—еӨҚдҪҚж”ҜжҢҒпјӣ
- **Sid**пјҡSID иҜ»еҸ–ж”ҜжҢҒпјӣ
- **Jtag**пјҡJTAG и°ғиҜ•дҪҝиғҪж”ҜжҢҒпјӣ
- **DDR**пјҡDDR жҺ§еҲ¶еҷЁеҲқе§ӢеҢ–ж”ҜжҢҒпјӣ
- **SPI Nor Flash**пјҡSPI NOR Flash ж“ҚдҪңж”ҜжҢҒпјӣ
- **SPI Nand Flash**пјҡSPI NAND Flash ж“ҚдҪңж”ҜжҢҒгҖӮ
