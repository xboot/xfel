# write32

向设备内存中写入一个 32 位的值。

## 用法

```shell
xfel write32 <address> <value>
```

## 参数

| 参数 | 说明 |
|------|------|
| `<address>` | 要写入的内存地址 |
| `<value>` | 待写入的 32 位值 |

## 说明

向指定地址写入 4 字节数据，常用于配置寄存器或修改内存标志。

:::note
FEL 协议仅支持按字节访问，该命令内部通过字节写操作实现。
:::

## 示例

向地址 `0x01c20000` 写入 `0x80000000`：

```shell
xfel write32 0x01c20000 0x80000000
```
