# write

将本地文件写入设备内存，写入过程带进度显示。

## 用法

```shell
xfel write <address> <file>
```

## 参数

| 参数 | 说明 |
|------|------|
| `<address>` | 内存起始地址 |
| `<file>` | 待写入的本地文件路径 |

## 说明

该命令读取本地文件的完整内容并写入设备内存的指定地址，传输过程会显示进度条，是加载固件到内存最常用的命令。

## 示例

将 `u-boot.bin` 写入内存地址 `0x81700000`：

```shell
xfel write 0x81700000 u-boot.bin
```
