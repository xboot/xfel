# read

从设备内存读取数据并保存到文件，读取过程带进度显示。

## 用法

```shell
xfel read <address> <length> <file>
```

## 参数

| 参数 | 说明 |
|------|------|
| `<address>` | 内存起始地址 |
| `<length>` | 读取的字节数 |
| `<file>` | 保存数据的文件路径 |

## 说明

该命令将指定长度的内存数据读取并写入本地文件，传输过程会显示进度条，适合读取较大块的数据。

## 示例

从地址 `0x81700000` 读取 1MB 数据到 `dump.bin`：

```shell
xfel read 0x81700000 0x100000 dump.bin
```
