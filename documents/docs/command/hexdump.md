# hexdump

以十六进制格式转储指定内存区域。

## 用法

```shell
xfel hexdump <address> <length>
```

## 参数

| 参数 | 说明 |
|------|------|
| `<address>` | 内存起始地址，支持十进制或 `0x` 前缀的十六进制 |
| `<length>` | 读取的字节数 |

## 说明

该命令读取设备内存并以带地址、十六进制值与 ASCII 字符的格式输出到终端，便于直观查看内存内容。

## 示例

转储起始地址 `0x80000000` 处的 256 字节：

```shell
xfel hexdump 0x80000000 256
```
