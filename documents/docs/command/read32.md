# read32

从设备内存中读取一个 32 位的值。

## 用法

```shell
xfel read32 <address>
```

## 参数

| 参数 | 说明 |
|------|------|
| `<address>` | 要读取的内存地址 |

## 说明

读取指定地址处的 4 字节数据并以十六进制形式打印。常用于查询寄存器值或验证内存内容。

:::note
FEL 协议仅支持按字节访问，该命令内部通过字节读操作实现。
:::

## 示例

```shell
xfel read32 0x01c20000
```

输出示例：

```
0x00000000
```
